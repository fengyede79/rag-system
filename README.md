# my-rag

这是一个独立发布的 `my-rag` 项目仓库。

## 目录说明

- `code/C8/`: 项目代码主目录
- `data/C8/`: 项目数据目录

## 主要入口

- `code/C8/README.md`: 项目详细说明
- `code/C8/main.py`: 主程序入口
- `code/C8/web_app.py`: Web 应用入口
- `code/C8/requirements.txt`: Python 依赖

## 说明

为了避免把本地敏感信息和生成文件上传到 GitHub，仓库已排除以下内容：

- `.env`
- `__pycache__/`
- `.pytest_cache/`
- `vector_index/`
- `evaluation/latest_report.json`
